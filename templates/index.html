Help Modal

Usefull Commands (Speech Recognition must be on)
Command Description
start system Starts the system.
stop system Stops the system.
detect people Begins live people detection.
detect emotion Begins live emotion detection.
detect move Begins live movement detection.
start dancing Makes the robot start dancing.
help help Stops the robot's movements and disables options on the screen in case of danger.
start game or play again Starts a rock-paper-scissors game against the robot.
tv mode Switches the robot to TV mode.
pong Makes the robot play the Pong game.
Turn off Speech Recognition if you want to disable these commands
